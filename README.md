# Books Directory

## Run Locally

Install Node.js [Link](https://nodejs.org/en/)

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd books
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

Start json server

```bash
npm run server-start
```
